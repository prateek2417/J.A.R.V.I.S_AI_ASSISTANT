# J.A.R.V.I.S_AI_ASSISTANT
Jarvis is a Python-based virtual assistant powered by Google's Gemini 2.5 Pro model. It uses voice commands to perform tasks like playing music, reading real-time news headlines, and opening websites. Bridging the gap between automation and AI, Jarvis offers a hands-free experience for daily tasks using libraries like SpeechRecognition and pyttsx3.
